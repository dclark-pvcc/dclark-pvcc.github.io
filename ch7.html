Chapter 7: Computer Networks & Cloud Computing

Dylan Clark
Briefly describe or define:
  1. computer network: A collection of independent computer systems connected together by telecommunication links, made for sharing information and resources.
  2. analog device: The voice-oriented dial-up telephone network is an example of an analog device, which are good at transmitting human voices, which vary in pitch and volume.
  3. digital device: A digital subscriber line (DSL) trasnmits digital signals, which are a sequence of 0s and 1s.
  4. modem: A device that modulates or alters a carrier, which is a standard analog signal, and encodes it into binary information.
  5. bandwidth: The capacity for transmitting data.
  6. ethernet: The most widely used broadband technology.
  7. WLAN: (wireless local area network) A wireless network that users can connect to locally, within a few hundred feet.
  8. IEEE 802.11 wireless network standard: Another name for Wi-Fi.
  9. MAN: (metropolitan area network) A larger scale WLAN, typically covering a neighborhood in size.
  10. Internet of Things: The concept of everyday objects having connection to the internet.
  11. LAN: A local area network which connects devices within close proximity.
  12. repeater: Amplifies and forwards a signal.
  13. WAN: Wide Area Networks connect devices that aren't in close proximity.
  14. packet and packet switching: An information block transmitted through the network as a single unit, which hop from nodes to reach thier destination.
  15. ISP: An internet service provider is a business which provides private access to their network.
  16. firewall: A software or hardware component that controls access from a network to a computer system.
  17. protocol: When it comes to networking, it is widely agreed upon rules, conventions, and agreements to make the exchange of information efficient and orderly.
  18. cloud computing: A computing system where data is stored and services are provided outside the view of the user.
  19. net neutrality: The idea that an Internet service provider (ISP) should treat all users, platforms, and content equally.
Briefly describe the five layers of TCP/IP (the internet protocol hierarchy) :
  1. Layer 1. Physical layer: Governs the exchange of binary digits across a physical communication channel, such as a cable, wire, or radio.
  2. Layer 2. Data link layer: Address and solve these error handling and framing.
  3. Layer 3. Network layer: Makes sure that a message is delivered from the original site to it's destination.
  4. Layer 4. Transport layer: "program-to-program" delivery service which moves messages from one specific program to another.
  5. Layer 5. Application layer: The rules for implementing the end-user services provided by a network.
List and briefly describe at least 3 network services and benefits:
  1. Email, which is electronic mail, allowing for users to message one another no matter how distant.
  2. Resource sharing, which allows users to share physical resources, such as a printer or storage device, as well as logical resources, such as software, data, and information.
  3. Electronic commerce, which allows for paperless exchanges of goods and services.
