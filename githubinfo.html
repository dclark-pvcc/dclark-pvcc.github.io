All About Github

Dylan Clark
What is a version of control system?
  • A safety net for recovering a projects files.
What is Git?
  • A free and open-source VCS.
What is Github?
  • A cloud platform built on Git.
What is the difference between Git and Github?
  • Github is a platform that provides hosting for Git repositories.
Who started Github and how was it started?
  • Tom Preston-Werner, Chris Wanstrath, P.J. Hyett, and Scott Chacon created Github.
  • The concept came about from issues with Git, as it was difficult to use for collaborative development.
What company owns it now?
  • Microsoft
How much does a Github account cost?
  • Free for personal/organizational purposes, however paid team accounts cost $4 per user per month.
What is the Octocat?
  • GitHub's mascot which is one of the tech industry's most recognizable symbols.
