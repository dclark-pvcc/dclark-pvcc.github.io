Briefly describe or define:
  1. Information Security: The goal of keeping information protected from those who do not have permission to access it.
  2. Authentication: The process of verifying who has the right to gain access to the computer.
  3. Authorization: Governs what an authenticated user of a computer is allowed to do.
  4. Dual Authentication: A second authentication measure after the user enters the user ID and password.
  5. Social Engineering: The process of using people to get the information you want.
  6. Virus: A computer program that embeds itself within another program or file in order to infect a host computer and then spread.
  7. Worm: Similar to a virus, but can send copies of itself to other nodes on a computer network without having to be carried by an infected host file.
  8. Trojan Horse: A computer program that does some legitimate computational task but, unbeknownst to the user, also contains code to perform malicious attacks.
  9. Ransomware: Malware that invades a user's computer, encrypts all the files, then demands a ransom payment for the files to be released.
  10. Phishing: The practice of sending emails purporting to be legimate to a large number of potential victims in the hopes of luring users to visit a fake website, where personal information can be collected.
  11. White Hat: The security experts who try to keep us safe in cyberspace
  12. Black Hat: The bad people who try to ferret out computer system weaknesses for financial gain, control, or outright destruction.
  13. Encryption/Decryption: Encryption is to make information meaningless even if someone does manage to steal it, while Decryption is the reverse.
  14. Cryptography: The science of secret writing.
  15. Symmetric Encryption Algorithm: An encryption algorithm that requires a secret key shared by sender and receiver.
  16. Caesar Cipher: An encryption algorithm that shifts each character in the message to another character some fixed distance farther along in the alphabet.
  17. Block Cipher: Any encryption algorithm that encodes a block of characters together, so that each coded character is the result of several plaintext characters.
  18. Data Encryption Standard (DES): A symmetric encryption algorithm developed in the 1970s but still widely used.
  19. Public Key System: The encryption key for messages to go to a particular receiver is broadcast to everyone, but the decryption key cannot be derived from it and is known only by the receiver.
  20. Embedded Computers: Computational devices such as chips, processors, or computers that are embedded within another system.

List six ways to protect yourself from most cyber attacks (see page 406):
  1. Be sure your computer has up-to-date, reputable antivirus software.
  2. Be sure your computer has up-to-date, reputable firewall software.
  3. Be sure your computer has up-to-date, reputable antispyware.
  4. Set your browser to use a pop-up blocker.
  5. Always install the latest security patches or updates to your operating system.
  6. Don't send personal or financial information in response to any email
